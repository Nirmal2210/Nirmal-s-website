Study Distraction Monitor Dashboard with Analytics

Overview

A Django-based web application that monitors student attention and distraction during study sessions using real-time webcam AI detection. The system combines face tracking, eye tracking, and head position analysis with machine learning to predict focus/distraction states.

Key Features

üé• Real-Time Detection

Face detection, eye tracking with EAR (Eye Aspect Ratio), and head orientation analysis. Real-time alerts if face/eyes missing for >5 seconds.

üìä Session Tracking

Monitors tab switching, keyboard activity, mouse movements, and idle time detection during study sessions.

ü§ñ Machine Learning

Random Forest classifier with 100% accuracy on training data. Real-time confidence scores for focus/distraction predictions.

üìà Analytics & Reports

Weekly attendance charts, session summaries, attention percentages, and CSV export for external analysis.

Technical Stack

Django Python OpenCV Scikit-learn Face Recognition Eye Tracking Machine Learning JavaScript HTML/CSS

Detailed Features

Real-Time Detection System

  • Face Detection: Detects if user is present in front of screen
  • Eye Tracking: Monitors if eyes are open/closed using Eye Aspect Ratio (EAR)
  • Head Position: Detects head orientation (left, center, right)
  • Attention Warnings: Real-time alerts if face/eyes missing for >5 seconds

Session Navigation & Analytics

  • Tab Switch Monitoring: Counts when user switches browser tabs
  • Keyboard Activity: Tracks key presses during session
  • Mouse Tracking: Monitors mouse movements and gestures
  • Idle Time: Measures inactivity periods

Machine Learning Component

  • Model Type: Random Forest Classifier
  • Training Accuracy: 100% on training data
  • Prediction Type: Focus/Distraction classification
  • Real-time Probabilities: Confidence scores for predictions

Reports & Export

  • Weekly Attendance Chart: Line chart showing daily attention percentage
  • Session Summary: Focused/Distracted session counts
  • Attention Percentage: Average attention rate over 7 days
  • CSV Export: Download all session data for external analysis

Technologies Used

  1. Backend: Django 6.0+, Python 3.8+
  2. Computer Vision: OpenCV for face/eye detection
  3. Machine Learning: Scikit-learn (Random Forest), Pandas, NumPy
  4. Model Serialization: Joblib
  5. Frontend: HTML, CSS, JavaScript
  6. Data Storage: SQLite Database
